LSI, A brief overview:

Latent Semantic Analysis, or LSA, is one of the foundational techniques in topic modeling.

The core idea is to take a matrix of what we have a document-term matrix and we decompose it into a separate document-topic matrix and a topic-term matrix.

You can find LSI algorithm implementation here.

 

NMF, A brief overview:

Non-negative Matrix Factorization is a Linear-algebraic model, that factors high-dimensional vectors into a low-dimensionality representation.

By factoring them into the lower-dimensional form, NMF forces the coefficients to also be non-negative 

You can find NMF algorithm implementation here.

 

CORPUS AND IMPLEMENTATION DETAILS:

Here, we used a Medical text-corpus which consists of around 1000 text files containing text about medical disorders, phenomenon etc.

Also, both Latent Semantic Analysis and Non-Negative Matrix Factorization algorithms are implemented in python using the libraries Gensim and NLTK
